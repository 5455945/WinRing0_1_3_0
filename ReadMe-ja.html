 WinRing0 は Windows 用のハードウェアアクセスライブラリです。

-------------------------------------------------------------------------------
 主な機能
-------------------------------------------------------------------------------
 - I/O ポート
 - MSR (Model-Specific Register)
 - PCI バス
 にアクセスする機能などを提供します。詳細は付属マニュアルをご覧ください。

-------------------------------------------------------------------------------
 動作環境
-------------------------------------------------------------------------------
 OS : [x86] 2008/Vista/2003/XP/2000/NT4/Me/98
      [x64] 2008/Vista/2003/XP
